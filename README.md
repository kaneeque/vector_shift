# Vector Shift – Full Stack Assessment

A full-stack assessment project showcasing a clean frontend–backend architecture, modern UI handling, and API-driven design with production-ready Git practices.

---

## Overview

This repository contains a complete frontend and backend implementation developed as part of a technical assessment.  
The project emphasizes **code clarity**, **scalable structure**, and **real-world development standards**.

---

## Tech Stack

### Frontend
- React
- JavaScript (ES6+)
- HTML5, CSS3
- Node.js & npm

### Backend
- Python
- FastAPI
- Uvicorn

## Setup & Installation

### Prerequisites
- Node.js (v16 or higher)
- Python 3.10+
- Git

## Backend Setup
- cd Backend
- python -m venv venv
- venv\Scripts\activate
- pip install -r requirements.txt
- uvicorn main:app --reload

## Frontend Setup
- cd Frontend
- npm install
- npm start
